Light on Haṭha Yoga — local working copy
========================================

HOW TO OPEN
-----------
Double-click  index.html  (opens in any browser).
Everything runs locally, offline. No server or internet needed.

WHAT'S INSIDE
-------------
index.html        The landing page — two entrances:
                    • HYP — The Haṭha Yoga Project  (content migrating, placeholder for now)
                    • HP  — Haṭhapradīpikā          (opens the full digital reader)

hp/               The Haṭhapradīpikā critical-edition reader (complete, 553 verses:
                  Devanagari · IAST · translation · philological commentary).
                  Captured from the project's build-0.1a. The five asana media
                  clips it links to (resources/*.mp4) were not on the source
                  server, so they are not included; add them later if wanted.

hyp/              Placeholder page for the Haṭha Yoga Project. The real content
                  can't be captured until hyp.soas.ac.uk (or a temporary URL)
                  is back up. Once it is, it drops in here next to hp/.

Destination for the live version: yoga.ames.ox.ac.uk
